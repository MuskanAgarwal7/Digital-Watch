# ⏰ Digital Clock

A real-time digital clock built using **HTML, CSS, and JavaScript**.

## 🚀 Features
- ⏳ Updates every second.
- 🌙 Dark-themed background.
- 📱 Fully responsive.
- 🕰️ 12-hour format (AM/PM).
